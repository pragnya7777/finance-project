# Finance Project 
 
## Project Description 
Finance Project is a full-stack personal finance management application. 
It helps users track expenses, manage budgets, and view wallet balances. 
Web version: React, runs in browser 
Mobile version: React Native with Expo, runs on Android/iOS 
 
## Project Structure 
finance-project/ 
├─ FinanceApp/          # Expo mobile app 
│  ├─ App.js 
│  ├─ components/      # Dashboard, Wallet, BudgetPredictor 
│  └─ package.json 
├─ frontend/            # React web app 
│  ├─ src/ 
│  │  ├─ components/   # Dashboard, Wallet, BudgetPredictor 
│  │  └─ App.js 
│  └─ package.json 
└─ README.md            # This file 
 
## Features 
- Dashboard: View financial status 
- Wallet: Track balance and transactions 
- Budget Predictor: Set budgets and get insights 
- Responsive UI for web and mobile 
- Cross-platform: Single codebase 
 
### Web Version 
cd frontend 
npm install 
npm start 
### Mobile Version (Expo) 
cd ../FinanceApp 
npm install
